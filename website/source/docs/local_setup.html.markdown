# Local Setup
Arc and the Lyra Cli are developed on [Go](https://golang.org/). In order to able to make any changes the first thing to do is setup Go and related project structures.

## Installation of Go

Install Go using brew:

    brew install go

We are actually using version `go1.6.2`.
